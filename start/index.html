這只是開始...

恭喜你看到了這邊，你已經對 Django 有了最基本的認知。那麼接下來我們該怎麼繼續下去呢？

  • Django Girls 問卷 - 如果今天 Workshop 有引起你對 Django 的一點興趣，我們之後將會舉辦 Django Girls 活動，這邊需要大家的一些意見 :)
  • 在原本的 Excuse Model 裡面加上一個 author 的欄位，記錄是誰講的
  • 把這個 example 從頭再做一次，看看自己記得多少
  • 目前我們的 CSS 都是寫在 HTML 當中，試著把 CSS 抽出去
  • 直接從現有的基礎，試著挑戰做個 Blog 或是留言板吧

建議也可以看看以下的參考資料：

  • Django Tutorial for PyCon APAC 2014 - 這是我在今年 PyCon 開的 Django Tutorial 講義
  • Writing your first Django app - Django 官方的 Tutorial
  • Django 官方文件 - 必看！文件相當詳細，不過建議有問題再來找。
  • Django Book - 雖然書的內容有點舊，不過還是有些東西值得參考
  • Getting Started with Django - 如果你比較喜歡看 video，或許可以參考這個網站
  • Two Scoops of Django - 近期寫的最好的 Django 書之一，有看一定有收獲
  • Django Packages - 想要幫網站加點其他功能嗎？可以先來這邊看看有沒有人有做好的 Django APP 可以用！

除此之外，也歡迎大家可以來 Taipei.py 的活動跟大家進行一些實體的交流，彼此之間多聊聊才更能夠有進步 :)
